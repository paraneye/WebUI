CWA/CWP

CWA and CWP List

CWAs and CWPs are high level WBS (Work Breakdown Structure) element. CWAs and CWPs are fundamental data required to begin project planning and budgeting. A CWA can be broken down to several CWPs by disciplines.

Adding New CWA and CWP

Selecting Add New displays pop-up window for adding a new CWA and its CWPs. Fields denoted with '*' are required fields.

  • CWA Number : CWA number
  • Description : description of CWA
  • CWP Number : CWP number
  • Discipline : select discipline that the CWP is associated with
  • Description : description of CWP

Editing CWA and CWP

Selecting CWA number or CWP number from the list pops up Edit window. All fields are editable.

Deleting CWA and CWP

Put a check to the CWA number or CWP number you want to delete and select Delete on the list screen. Deleting CWA removes all CWPs under it.
