Cost Code Assignment

Components List

You can assign cost codes to components at one of three menus, MTO, Import MTO, and Cost Code Assignment. In other words, cost codes assigned to components can be changed at any of these three menus. In Cost Code Assignment page, you can assign cost codes to components filtered by following classification.

  • CWP : CWP number. Each CWP has its associated discipline
  • Task Category : task categories under the discipline of the CWP are listed. Select a task category to classify the components.
  • Task Type : task types under the selected task category are listed. Select a task type to classify the components.
  • Progress : progress steps under the selected task type are listed. Select a progress step to classify the components.

Assigning Cost Code to Components

Put a check to components you want to assign a cost code and select Assign Cost Code. On the following pop-up window, you can select a cost code to assign to the selected components.

Assigning Cost Code to All

To assign a cost code to all components in the list, select Assign Cost Code to All. On the following pop-up window, you can select a cost code to assign to all components.
