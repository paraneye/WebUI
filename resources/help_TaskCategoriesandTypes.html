Task Categories and Types

Task Categories and Task Types List

All task categories and task types under each task category defined are listed in a form of tree. These task categories and types are used to classify materials in the library. There are predefined task categories and task types in default. In a tree list, there are three levels of nodes. Nodes of first level are disciplines, nodes of second level are task categories, and nodes of third level are task types.

Adding New Task Category and Task Type

Select a node name (discipline or task category) from the tree list and then select Add New. Empty field for new node name appears. New node is generated under the selected node. Task type node cannot have any lower level of node.

Editing Task Category and Task Type

Select a node name (discipline, task category, or task type) from the tree list and then select Rename. You can change the name of the selected node.

Deleting Task Category and Task Type

Select a node name (discipline, task category, or task type) from the tree list and then select Delete.

Assigning Progress Type

Progress types are supposed to be assigned to materials in the library. Progress types are milestones of material installation. Assigning a progress type to materials makes it easy to track the progress of material installation work. By assigning a progress type to either task category or task type, you can assign the progress type to all the materials that are classified into the selected task category or task type.
