IWP Viewer

Viewing IWP

The contents of the selected IWP are listed by types. You can select an IWP you want to see by choosing for followings.

  • CWP : CWP which includes the IWP you want to see
  • IWP Status : status of IWP
  • IWP : IWP number

Printing IWP

If you select each of the contents IWP has, appropriate file viewer launches and displays it. You can use Print function of the file viewer to print it. To print all of contents of the IWP, select Print All.
