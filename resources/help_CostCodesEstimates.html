Cost Code Estimates

Cost Codes Estimates List

Cost codes that are defined at Cost Codes under Global Settings are listed. For project cost management, you need to enter estimates to the cost codes and the estimates become project¡¯s baseline costs. The cost codes with estimates are considered as project cost codes. Cost Codes must be defined in advance at Cost Codes under Global Settings.

Assigning Estimates to Cost Codes

Selecting Cost Code from the Cost Codes list displays pop-up window for entering estimates for the selected cost code. Fields denoted with '*' are required fields.

  • UOM : unit of measurement for the cost code
  • Estimated Quantity : quantity estimated to require for the cost code until the end of the project
  • Estimated Man-hour : man-hours estimated to require for the cost code until the end of the project

Editing Cost Code Estimate

Selecting Cost Code from the Cost Codes list pops up Edit window.

Import File

You can add bulk cost codes estimates by importing excel file. Before starting to import cost codes information, you need to download template file and fill it out with cost codes estimates information. You can download a template file from Import File popup window. In the template file, column titles denoted with '*' are required fields. Notes in Readme sheet will help when you fill the template out.

  • Clicking Browse button opens pop-up windows asking for choosing a excel file from local computer.
  • To see what information is needed, download a excel template file. Fields denoted with '*' in it are required fields.
  • Selecting Save button imports cost codes information from the excel file and then shows the result

Import History

After importing is done, import results and logs are available. You can see which item failed for which reason. Downloading the list of failures as an excel file can help make corrections on those failures and then import it again.

  • Clicking Download shows import failures and fail reasons. You can download failures as a form of excel file.
  • To delete import logs, put a check to import logs you want to delete, and select Delete button
