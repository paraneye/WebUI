Document Library

Documents List

This menu gives the workspace to catalog and track all project related documents. This includes guidelines, specification, standards, procedures, plans, checklist, and all other documents for reference. It offers document version control by allowing to add versions and to download a specific version. Documents can be filtered by Document Type and searched with keyword for document Title and Description. All documents are available at mobile applications.

Adding New Document

Selecting Add New displays pop-up window for adding a new document. Fields denoted with '*' are required fields.

  • Title : title of the document
  • Document Type : select a document type
  • Description : description of the document
  • Choose a File : choose a file to upload from a local computer. File must be PDF format. But for Cover Page document type, it must be an image format

Updating Document

Selecting a document title in the documents list pops up Edit window. On the following Edit window, you can change description, upload a new version of document file, view version history, and download certain version of document file.

Deleting Document

Put a check to the document you want to delete in the document list and select Delete.
