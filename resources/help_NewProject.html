Creating New Project

Administrator only can create new project. Fields denoted with '*' are required fields.

  • Basic Information
    • Disciplines : select all disciplines applicable for the project
    • Project Name : project name
    • Project ID : project ID
    • Project Manager : select from users
    • Location : location where construction occurs
    • Logo : upload an image file of project logo
    • Description : description of the project
  • Client Information
    • Client : all companies that are added at Companies menu under Global Settings are listed in the list box. If you cannot find the company you want to choose, you can add it to the list of companies at Companies menu under Global Settings.
    • Client Project Name : client¡¯s project name
    • Client Project ID : client¡¯s project ID
  • Subcontractors
    • Subcontractors : companies that are added as a subcontractor type at Companies menu under Global Settings are listed in the list box. You can add multiple companies.

  • Assigned project manager can configure Settings and Data for the project.
