Progress Types

Progress Types List

All progress types defined are listed. These progress types are supposed to be assigned to materials in the library. This defined progress types are milestones of material installation.

Adding New Progress Type

Selecting Add New displays pop up window for adding a new progress type. Fields denoted with '*' are required fields.

  • Progress Type : progress type name
  • Progress Description : description of progress type
  • Add Progress Step : progress steps consisting of the progress type
    • Progress Step : progress step name
    • Weight (%) : weight value for the progress step in percentage.
    • Multiplied by Qty. : indicates if the estimated man-hour of material having the progress step has to be multiplied by the quantity of materials.
  • Sum of weights of all the progress steps must be 100

Editing Progress Type

Selecting progress type from the progress types list pops up Edit window. All fields are editable.

Deleting Progress Type

Put a check to progress type you want to delete and select Delete on the progress types list screen.
