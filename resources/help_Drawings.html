Drawings

Drawings List

All drawings available for the project are listed. Each drawing is associated with a CWP.This screen has three portions of functions:

  • Adding drawing information by Add New or Import File
  • Adding drawing image file by Upload Drawing
  • Binding drawing image file to its drawing information by Bind Drawings

  • Please be noted that drawing information and drawing image are added separately, and later bound each other.

Adding New Drawing

Selecting Add New displays pop-up window for adding a new drawing. Fields denoted with '*' are required fields.

  • CWP : CWP number that the drawing is associated with
  • Drawing Number : drawing number
  • Revision : revision number of the drawing
  • File Name : the name of drawing file. PDF format
  • Title : title of the drawing
  • Type : drawing type
  • Description : description of the drawing
  • Reference Drawings : drawing numbers of the drawings that are used for reference
  • Detail Drawings : drawing numbers of the drawings that have the detail view of certain area of the drawing to add
  • Image : image file of the drawing. Choose an image file from local computer. The image filename must be the same with File Name but not with its format

Editing Drawing

Selecting a drawing number from the drawings list pops up Edit window. All fields are editable.

Deleting Drawing

Put a check to the drawing you want to delete and select Delete on the drawings list screen.

Import File

You can add bulk drawings by importing excel file. You can download a template file from Import File pop-up window. In the template file, column titles denoted with ¡®*¡¯ are required fields. Notes in Readme sheet will help when you fill the template out.

  • Clicking Browse button opens pop-up windows asking for choosing a excel file from local computer.
  • To see what information is needed, download a excel template file. Fields denoted with ¡®*¡¯ in it are required fields.
  • Selecting Save button imports drawings information from the excel file and then shows the result.

  • Drawing information import file has the file names of drawing images but importing it does not add actual image files of drawings. You must separately upload image files of drawings by Upload Drawing.

Import History

After importing is done, import results and logs are available. You can see which item failed for which reason. Downloading the list of failures as an excel file can help make corrections on those failures and then import it again.

  • Clicking Download shows import failures and fail reasons. You can download failures as a form of excel file.
  • To delete import logs, put a check to import logs you want to delete, and select Delete button.

Upload Drawing

You can upload image files of drawings as many as you want simply by dragging and dropping. Files must be PDF format and their names must be the same with the ones indicated in the drawing information so that the images of drawings are successfully bound to their drawing information

  • CWP : CWP number that are associated with the drawings to upload

Binding Drawings

After importing drawings information and uploading corresponding drawings image files are completed, you need to bind the drawing image files to the corresponding drawing information. Drawing image files that failed to be bound are called Orphan Drawings. Binding failures occurs when drawing image filename does not match the filename in any of drawing information. Drawing information without its bound drawing image file is Unbound Drawing Info.

List View Option

There are four types to view drawings list

  • ALL : all drawings are listed
  • Normal : drawings with bound image files are listed
  • Orphan Drawings : drawings that have only image files but not their drawing information are listed
  • Unbound Drawing Info : drawings that have only drawing information but not their image files are listed
