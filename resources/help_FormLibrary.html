Form Library

Form List

All forms generated with OZ designer are available. This menu offers version control by allowing to add versions and to download a specific version. Forms can be filtered by Form Type and searched with keyword for document Title and Description. All forms are available at mobile applications.

Adding New Form

Selecting Add New displays pop-up window for adding a new form. Fields denoted with '*' are required fields.

  • Title : title of the form
  • Form Type : select a form type
  • Description : description of the form
  • Choose a File : choose a file to upload from a local computer. File must be OZR or OZD format.

Updating Form

Selecting a form title in the forms list pops up Edit window. On the following Edit window, you can change description, upload a new version of document file, view version history, and download certain version of document file

Deleting Form

Put a check to the form you want to delete in the forms list and select Delete.
