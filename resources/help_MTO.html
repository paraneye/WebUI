MTO(Material Take Off)

The process of MTO is to take off materials from the drawings and to add additional information such as estimated cost and effort to install them. Materials taken off and estimated are called Components. The process of generating components is consisted of following three steps.

  • Finding a material from drawing
  • Searching Material Library for the material found in a drawing
  • Adding additional information to the material

Generating Components

  • Finding materials from drawings

    Choosing a CWP and then a drawing to take off materials from displays the selected drawing¡¯s image. Find materials to be taken off from the drawing. If the MTO is completed or in progress with the drawing, already generated components are listed at the bottom of the page.

  • Searching Material Library for materials

    Search Material Library for the materials you found from the drawing. To make it easy to search material library, following four data are needed for classification.

    • Discipline: select a discipline the material is associated with
    • Task Category: task categories under the selected discipline are listed. Select a task category to classify the material.
    • Task Type: task types under the selected task category are listed. Select a task type to classify the material.
    • Description: description of the material

  • Select the material you want from the search result and a new panel for adding more information to the material appears.

  • Adding additional information
    To finalize component generation, you need to add following information to the material searched from Material Library. Fields denoted with '*' are required fields.
    • Quantity : quantity of material to install
    • Description : description of the component to generate
    • Cost Code : cost code to use for the material
    • Man Hour : man hours required to install the amount of the material that is given in Quantity field. This is automatically calculated by the formula when the quantity is entered.
    • Foundation Number : number of civil foundation where the material is supposed to install
    • Engineering Tag : material tag number shown in the drawing
    • User Defined Fields : user can add user defined fields to enter additional information to the material
      • Title: name of the user defined field
      • Value: value of the user defined field
      • Visible: indicate whether the user defined field needs to be visible when mobile applications see the component

  • Selecting Save generates a new component

Viewing Components

Choosing a CWP and then a drawing displays the selected drawing¡¯s image. If the MTO is completed or in progress with the drawing, already generated components are listed at the bottom of the page

Editing Components

Selecting a component from the components list pops up Edit window.

Deleting Components

Put a check to components you want to delete in the components list and select Delete.
