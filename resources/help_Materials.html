Materials

Materials List

All materials saved in the library are listed. Materials can be filtered by the selection of discipline, task category and task type.

Adding New Material

Selecting Add New displays pop up window for adding a new material. Fields denoted with '*' are required fields.

  • Discipline : select a discipline the material is associated with
  • Task Category : task categories under the selected discipline are listed. Select a task category to classify the material.
  • Task Type : task types under the selected task category are listed. Select a task type to classify the material.
  • Description : description of material
  • Vendor : vendor of material
  • Part Number : part number of material
  • UOM : select unit of measurement for the material
  • Man Hour : estimated man hour to install the material
  • Cost Code : select cost code to pay for the material
  • Add User Defined Field : user can add user defined fields to enter additional information to the material
    • Title : name of the user defined field
    • Value : value of the user defined field

Editing Material

Selecting material from the materials list pops up Edit window. All fields are editable.

Deleting Material

Put a check to material you want to delete and select Delete on the material list screen.

Import File

You can add bulk material information by importing excel file. Before you import material information, you need to download template file and fill it out with material information. You can download a template file from Import File popup window. In the template file, column titles denoted with '*' are required fields. Notes in Readme sheet will help when you fill the template out.

  • Clicking Browse button opens pop-up windows asking for choosing a excel file from local computer.
  • To see what information is needed, download a excel template file. Fields denoted with '*' in it are required fields.
  • Selecting Save button imports data from the excel file and then shows the result.

Import History

After importing is done, import results and logs are available. You can see which item failed for which reason. Downloading the list of failures as an excel file can help make corrections on those failures and then import it again.

  • Clicking Download shows import failures and fail reasons. You can download failures as a form of excel file.
  • To delete import logs, put a check to import logs you want to delete, and select Delete button.
