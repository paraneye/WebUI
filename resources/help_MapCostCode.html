Mapping Project Cost Codes to Client Cost Codes

You can map simply by dragging project cost code and dropping it onto its corresponding client cost code. You need to do this until no more project cost codes left. Typically, you may not be able to put them in a one to one correspondence because the number of client cost codes is not the same with the number of project cost codes. In other words, a client cost code can have more than one project cost codes.
