Cost Codes

Cost Codes list

All defined cost codes are listed. Cost codes are supposed to assign to all materials in the library and all the effort that need to be considered as cost.

Adding New Cost Code

Selecting Add New displays pop up window for adding a cost code. Fields denoted with '*' are required fields.

  • Cost Code : cost code name
  • Description : description of the cost code
  • Direct : indicate if the cost code is direct or indirect

Editing Cost Code

Selecting cost code from the cost codes list pops up Edit window.

Deleting Cost Code

Put a check to cost code you want to delete and select Delete on the cost codes list screen.

Import File

You can add bulk cost codes by importing excel file. Before starting to import cost code information, you need to download template file and fill it out with cost code information. You can download a template file from Import File popup window. In the template file, column titles denoted with '*' are required fields. Notes in Readme sheet will help when you fill the template out.

  • Clicking Browse button opens pop-up windows asking for choosing a excel file from local computer.
  • To see what information is needed, download a excel template file. Fields denoted with '*' in it are required fields.
  • Selecting Save button imports data from the excel file and then shows the result.

Import History

After importing is done, import results and logs are available. You can see which item failed for which reason. Downloading the list of failures as an excel file can help make corrections on those failures and then import it again.

  • Clicking Download shows import failures and fail reasons. You can download failures as a form of excel file.
  • To delete import logs, put a check to import logs you want to delete, and select Delete button.
