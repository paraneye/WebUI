Drawing Types

Drawing Types List

All drawing types defined are listed. These drawing types are used when you add a drawing for a project and choose a drawing type for it

Adding New Drawing Type

Selecting Add New displays pop up window for adding a new drawing type. Fields denoted with '*' are required fields.

  • Drawing Type : drawing type
  • Drawing Description : description of drawing type

Editing Drawing Type

Selecting drawing type from the drawing types list pops up Edit window. All fields are editable.

Deleting Drawing Type

Put a check to drawing type you want to delete and select Delete on the drawing types list screen
