Members

Members List

All members assigned to the project are listed. The members are Sigma users that were assigned to project roles based on project resource management plan. Since members are Sigma users they use Sigma services with their own Sigma user ID. On Role Hierarchy page, you can make a hierarchical arrangement among project members.

Adding New Member

Selecting Add New displays pop up window for adding a new member. Fields denoted with '*' are required fields.

  • Name : select the name you want to add
  • Discipline : select the disciplines that the member works on for project
  • Role : Role of the member. Assign multiple roles to a member

Editing Member

Selecting member name from the member list pops up Edit window

Deleting Member

Put a check to the member you want to delete and select Delete on the members list screen.
