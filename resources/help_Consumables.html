Consumables

Consumables List

All consumables saved in the library are listed.

Adding New Consumable

Selecting Add New displays pop up window for adding a new consumable. Fields denoted with ¡®*¡¯ are required fields.

  • Vendor : vendor of consumable
  • Description : description of consumable
  • Part Number : part number of consumable
  • UOM : select unit of measurement for the consumable
  • Add User Defined Field : user can add user defined fields to enter additional information to the consumable
    • Title : name of the user defined field
    • Value : value of the user defined field

Editing Consumable

Selecting consumable from the consumables list pops up Edit window. All fields are editable.

Deleting Consumable

Put a check to consumable you want to delete and select Delete on the consumables list screen.

Import File

You can add bulk consumable information by importing excel file. Before you import consumable information, you need to download template file and fill it out with consumable information. You can download a template file from Import File popup window. In the template file, column titles denoted with ¡®*¡¯ are required fields. Notes in Readme sheet will help when you fill the template out.

  • Clicking Browse button opens pop-up windows asking for choosing a excel file from local computer.
  • To see what information is needed, download a excel template file. Fields denoted with '*' in it are required fields.
  • Selecting Save button imports data from the excel file and then shows the result.

Import History

After importing is done, import results and logs are available. You can see which item failed for which reason. Downloading the list of failures as an excel file can help make corrections on those failures and then import it again.

  • Clicking Download shows import failures and fail reasons. You can download failures as a form of excel file.
  • To delete import logs, put a check to import logs you want to delete, and select Delete button.
