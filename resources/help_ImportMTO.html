Import MTO

Import MTO(Material Take Off)

Import MTO is to conduct the process of MTO at once. All information required for MTO must be contained in the excel file to import. This helps you avoid the repetition of doing MTO for all materials in all drawings

Import File

You need to get MTO information by importing excel file. Before you import MTO information, you need to download template file and fill it out with MTO information. You can download a template file from Import File popup window. In the template file, column titles denoted with '*' are required fields. Notes in Readme sheet will help when you fill the template out.

  • Clicking Browse button opens pop-up windows asking for choosing a excel file from local computer.
  • To see what information is needed, download a excel template file. Fields denoted with '*' in it are required fields.
  • Selecting Save button imports data from the excel file and then shows the result.

  • Components generated by importing MTO information file are listed.

Import History

After importing is done, import results and logs are available. You can see which item failed for which reason. Downloading the list of failures as an excel file can help make corrections on those failures and then import it again.

  • Clicking Download shows import failures and fail reasons. You can download failures as a form of excel file.
  • To delete import logs, put a check to import logs you want to delete, and select Delete button.

Viewing Components

Only after Import File is completed, generated components are listed. If you want to view components at other times, go to MTO under Settings.

Editing Components

Selecting a component from the components list pops up Edit window.

Deleting Components

Put a check to components you want to delete in the components list and select Delete.
