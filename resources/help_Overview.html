Menus for Project

Menus for project consist of four groups, Settings, Data, Tools, and Document Control. Once a project is created, project manager needs to configure project Settings and project Data as a preparation stage before project members begin project activities. To configure Settings and Data, project manager needs to make sure that all of settings under Global Settings were completed in advance.
