Equipment

Equipment List

All equipment saved in the library is listed. Equipment can be filtered by the selection of category 1, category2, and category3.

Adding New Equipment

Selecting Add New displays pop up window for adding new equipment. Fields denoted with '*' are required fields.

  • Category1 : select category1 to classify the equipment
  • Category2 : category2 items under the selected category 1 item are listed. Select a category2 item to classify the equipment.
  • Category3 : category3 items under the selected category 2 item are listed. Select a category3 item to classify the equipment.
  • Description : description of equipment
  • Spec : add additional information such as size, length, weight, width, height, capacity, volume and etc.
  • Type : add additional information such as performance type, purpose type, use type, installation type, and etc.
  • Add User Defined Field : user can add user defined fields to enter additional information to the equipment
    • Title : name of the user defined field
    • Value : value of the user defined field

Editing Equipment

Selecting equipment from the equipment list pops up Edit window. All fields are editable.

Deleting Equipment

Put a check to equipment you want to delete and select Delete on the equipment list screen.

Import File

You can add bulk equipment information by importing excel file. Before you import equipment information, you need to download template file and fill it out with equipment information. You can download a template file from Import File popup window. In the template file, column titles denoted with '*' are required fields. Notes in Readme sheet will help when you fill the template out.

  • Clicking Browse button opens pop-up windows asking for choosing a excel file from local computer.
  • To see what information is needed, download a excel template file. Fields denoted with '*' in it are required fields.
  • Selecting Save button imports data from the excel file and then shows the result.

Import History

After importing is done, import results and logs are available. You can see which item failed for which reason. Downloading the list of failures as an excel file can help make corrections on those failures and then import it again.

  • Clicking Download shows import failures and fail reasons. You can download failures as a form of excel file.
  • To delete import logs, put a check to import logs you want to delete, and select Delete button.
