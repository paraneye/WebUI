Schedule

You can make a project baseline schedule by copying an original P6 schedule and then updating it with additional project data generated by Sigma. This baseline schedule can be used to compare to actual progress as a project progresses. Project schedule is displayed. This page is consisted of two parts:

  • Imported from schedule : schedule information imported from P6 project schedule.
  • Projected by Sigma : schedule information projected by Sigma planning feature

Importing Schedule

To import an original P6 schedule for a project, select a P6 project name from the list. After importing is completed, WBS, Schedule Line Item, Start, Finish, and Duration of Imported from Schedule are visible.

  • WBS: item of Work Break Down
  • Schedule Line Item: activity item under WBS of P6 project schedule. P6 project schedule is planned by the level of SLI
  • Start: planned start date of the schedule line item
  • Finish: planned finish date of the schedule line item
  • Duration: planned duration of the schedule line item. This shows in hour.

Assigning CWP and General Foreman to Schedule

You can add and update information of Discipline, CWP, and Assign to by schedule line items. Selecting a schedule line item displays pop-up window for entering additional information

  • Discipline : discipline to associate with the schedule line item
  • CWP : CWP number to associate with the schedule line item
  • Assigned To : choose a General Foreman for the schedule line item

Projecting Schedule by Sigma

Once schedule line item gets assigned to CWP and General Foreman, it becomes ready to do Build Schedule with TrueTask mobile application. Build Schedule generates Start, Finish, Duration, Estimated Man Hours, and Assigned Crew of Projected by Sigma by packaging components to install by schedule line items.

  • Start : planned start date of the schedule line item generated by TrueTask
  • Finish : planned finish date of the schedule line item generated by TrueTask
  • Duration : planned duration of the schedule line item generated by TrueTask. This shows in hour.
  • Estimated Man Hours : man hours estimated by TrueTask to complete the schedule line item
  • Assigned Crew : the number of crew members estimated by TrueTask to complete the schedule line item for the planned duration

Deleting Schedule

Selecting Delete Schedule deletes entire project baseline schedule and all related project data. Administrator only can delete schedule.
